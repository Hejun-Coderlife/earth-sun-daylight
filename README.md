# 🌍 地球 · 太阳 · 日照模拟

一个可交互的网页，用 3D 地球模型 + 真实天文公式，解释**为什么不同纬度的白昼时间不一样**。

![type](https://img.shields.io/badge/单文件-纯前端-blue) ![tech](https://img.shields.io/badge/Three.js-3D-orange)

## ✨ 功能

**五种可切换模式（顶部按钮）：**

1. 🌗 **日照 · 昼夜** — 倾斜 23.5° 的自转地球、太阳平行光、可选纬度，实时算白昼长度、极昼/极夜。
2. 🌙 **地月系统** — 月球真实绕地球公转、潮汐锁定自转，实时显示月相与月龄。
3. 🔄 **公转 · 四季 · 距离** — 地球沿椭圆轨道绕日，实时显示日地距离变化、节气、近/远日点。
4. 🪐 **太阳系** — 八大行星各自绕日（周期符合开普勒定律）。
5. 🌌 **银河行进** — 太阳带着整个太阳系绕银心飞驰，行星划出螺旋拖尾。

**其它：**

- 🌐 **中英双语**切换（右上角 EN / 中文按钮）。
- 🌗 **亮/暗主题**切换。
- ⏯ 播放 / 暂停 + 速度滑块。
- 纬度对比图、原理讲解、天文公式 `cos(H) = −tan(φ)·tan(δ)`（本页实时计算）。

## 🚀 本地预览

直接用浏览器打开 `index.html` 即可（需要联网加载 Three.js 与地球贴图 CDN）。

## 🌐 部署到 GitHub Pages

1. 新建一个 GitHub 仓库，把本目录文件全部推上去：
   ```bash
   git init
   git add .
   git commit -m "地球太阳日照模拟"
   git branch -M main
   git remote add origin https://github.com/<你的用户名>/<仓库名>.git
   git push -u origin main
   ```
2. 仓库页面 → **Settings → Pages** → Source 选 `main` 分支、`/ (root)` 目录 → Save。
3. 等一两分钟，访问 `https://<你的用户名>.github.io/<仓库名>/` 即可。

> 入口文件名为 `index.html`，GitHub Pages 会自动识别为首页。

## 📐 原理一句话

地球离太阳远近几乎不变，真正的原因是**地轴倾斜 23.5°**：公转使太阳直射点在南北回归线间往返，把昼夜分界线「切歪」，纬度越高昼夜长短变化越剧烈，直至极昼极夜。
